# Duck-Tech
### A simulation of an organization for learning cybersecurity

Duck-Tech is a small-to-mid size organization that offers an SaaS solution business-to-business (B2B) for project management. Our employee counts are approximatly 50 and we have various considerations top be successful:

- We must protect the sensitive data of customers
- We must adhere to legal regulations
- We must have a flourishing and flexible network spanning across multiple departments
- We must establish secure remote connections for workers outside of the office which includes tenets like VPNs and Zero Trust.
- We need a public-facing web application for the overt operations of the business

## Departments with corresponding systems
- IT: AD/LDAP, DHCP, file server
- Dev/Engineering: Git Server, CI/CD pipeline, dev worksations
- Finance: Isolated VLAN, accounting workstation
- HR: Employee records server
- Sales/Marketing: CRM-like system, email server
- Executive: High-value targets for phishing
